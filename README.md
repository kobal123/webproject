WebShop 
===============

Beadandó a Magas Szintű Programozási Nyelvek 2 tárgyra.
-----


Technológiák:
-----

* Java
* Spring, Spring boot, Spring Security
* PostgreSQL
* Thymeleaf
* Javascript

Funckionalitások:
------

* Felhasználók regisztrálása
* Termékek létrehozása, kép hozzáadása
* Bevásárló kosár
* Termékhez komment hozzáadása
* Rendelés leadás
